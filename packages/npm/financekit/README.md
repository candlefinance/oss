   Financekit
</h1>

Wraps FinanceKit on iOS for consumption in React Native.

## Installation

To install `financekit` with **npm**:

```sh
npm install @candlefinance/financekit
```

To install `financekit` with **yarn**:

```sh
yarn add @candlefinance/financekit
```
